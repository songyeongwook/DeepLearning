# 프로젝트 소개

본 프로젝트는 딥러닝 및 머신러닝 기술을 학습하고 구현하기 위한 프로젝트입니다.

프로젝트를 통해:
- 데이터 전처리
- 딥러닝 모델 구현
- 신경망 학습 과정
- 모델 성능 분석
- 데이터 시각화

등을 실습하며 AI 모델의 동작 원리를 학습하였습니다.

# 프로젝트 구조

```plaintext
DeepLearning/
│
├── dataset/
│   └── 학습 데이터셋
│
├── model/
│   └── 딥러닝 모델 구조 및 학습 코드
│
├── train/
│   └── 모델 학습 코드
│
├── result/
│   └── 학습 결과 및 출력 데이터
│
├── util/
│   └── 데이터 처리 및 보조 기능
│
├── 결과 보고서/
│   └── 프로젝트 결과 보고서 및 분석 자료
│
├── README.md
│
└── requirements.txt
```
